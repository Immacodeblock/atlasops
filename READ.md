# AtlasOps

AtlasOps is a multi-tenant operations and compliance platform built as a portfolio project.

## Tech stack
- Azure Functions (.NET 8, isolated)
- Azure SQL Database
- Azure Static Web Apps
- GitHub Actions for CI/CD

## Architecture
- Serverless backend (Functions, Consumption plan)
- Relational data model with tenant isolation
- Cost-controlled Azure setup using free tiers

This project is designed to demonstrate senior-level system design, cloud architecture,
and operational best practices.
